# Diario Inteligente 

Asistente diario que usa inteligencia artificial para generar un resumen personalizado cada día.

## Qué incluye

- Clima
- Noticias destacadas
- Resultados de Universidad de Chile
- Datos financieros
- Frase reflexiva o emocional del día

## Tecnologías

- Python
- OpenAI API
- Jupyter Notebook
